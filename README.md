Proyecto de la escuela secundaria ESETP(Escuela Secundaria Especializada en Tecnico Profesional creo que significa) N°703 José Toschke de la ciudad de Puerto Madryn.
Este proyecto final es del area de electronica y el grupo a relizar el proyecto está conformado por Lazcano Iñaki, Juan Gonzalez y Camina Santiago.
El proyecto basicamente es un globo metereologico, que va a tener una carga con el objetivo de medir datos atmosfericos(humedad, temperatura, presión, altura, longitud, latitud, 
hora UTC, fecha, radiación) y transmitirlos en tiempo real a traves de un modulo LoRa a una estación terrena, para el posterior analisis y mostrado de los datos, a traves de una 
pagina web en los que se podran ver los datos medidos y la ubicacion del globo.
